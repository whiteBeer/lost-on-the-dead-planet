## A little multiplayer game.

**Run local:**
1. cd backend
2. create .env file
```
 NODE_ENV=local
 APP_PORT=7789
```
3. npm i
4. npm start
5. cd ../frontend
6. npm i
7. npm run dev

**Client:** http://localhost:7790/

**Backend root:** /backend/server.ts

**Frontend next.js app root:** /frontend/src/app/layout.tsx

**Frontend pixi.js app root:** /frontend/src/pixi-app/App.ts
